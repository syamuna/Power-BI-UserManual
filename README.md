# Power-BI-UserManual
Power BI Dashboard User &amp; Setup Guide:

This document provides detailed instructions to connect Power BI Desktop to an on-premises SQL Server database, create a live dashboard, and publish it to Power BI Service for organizational sharing.
It is intended for data analysts, business users, and reporting administrators who want to build interactive dashboards with real-time data.

#Simple User Guide:
This document provides detailed information about dashboards and Power BI setup. It has been prepared in a way that makes it easy for non-technical users to understand.
